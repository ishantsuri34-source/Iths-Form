A modern and fully responsive Admission Portal developed for the Institute of Translational Health Sciences (ITHS), under Rayat Bahra University.

This project is designed to provide prospective students with structured information about the 2026 admissions cycle, available programs, eligibility criteria, and application guidance in a clear and user-friendly interface.

The portal focuses on clean UI/UX principles, responsive layouts, and smooth navigation to ensure accessibility across all devices including desktops, tablets, and smartphones.
